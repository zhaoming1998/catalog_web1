## 项目概述

## 网站分区

| 分区 | 说明 |
|------|------|
| 首页 Hero | 全屏背景 + 科技感粒子/网格/光晕动画，品牌口号展示 |
| 数据统计 | 带动画的数字计数器（行业深耕、产品型号、无尘车间、服务客户） |
| 产品目录 | 401/405两个系列，可折叠展开，支持系列筛选、点击查看大图(Lightbox) |
| 核心优势 | 品质管控、研发创新、智能制造、全球视野 |
| 生产实力 | 工厂/车间/设备图片展示 |
| 关于我们 | 公司简介 + 特色认证（ISO9001、7S管理、无尘车间、智能自动化） |
| 联系我们 | 联系信息 + 询盘表单 |
| 页脚 | 快速链接、产品系列、联系方式 |

## 主要功能

- **响应式设计**：适配桌面、平板、手机
- **导航栏吸顶**：滚动时背景变白、毛玻璃效果
- **产品系列折叠**：默认收起，点击"展开产品图库"按钮平滑展开，显示产品数量
- **系列筛选**：全部系列 / 401 / 405 快速切换
- **图片灯箱**：点击产品图片弹出大图，支持左右翻页和键盘操作
- **滚动动画**：元素进入视口时淡入
- **返回顶部按钮**
- **数字计数动画**

## 新增产品系列

后续新增系列只需以下步骤：

1. 在 `assets/images/products/` 下新建文件夹（如 `501/`）
2. 将产品图片复制到该文件夹
3. 在 `index.html` 中参照 401/405 的结构添加新的 `product-series` 区块
4. 在 `main.js` 中添加对应的 `generateFileList501()` 函数并在 `renderProducts()` 中调用
5. 在筛选按钮区域和联系表单的下拉选项中添加新系列
