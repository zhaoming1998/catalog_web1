## 项目概述

## 网站分区

| 分区 | 说明 |
|------|------|
| 首页 Hero | 全屏泵头产品背景 + Ken Burns 缓慢缩放 + 科技感网格线/粒子漂浮/光晕呼吸动画 |

| 关于我们 | 公司简介 + 特色认证（ISO9001、7S管理、无尘车间、智能自动化） |
| 联系我们 | 联系信息 + 询盘表单 |
| 页脚 | 快速链接、产品系列、联系方式 |

## 主要功能

- **响应式设计**：适配桌面、平板、手机
- **导航栏吸顶**：滚动时背景变白、毛玻璃效果，自动高亮当前分区
- **产品系列折叠**：默认收起，点击"展开产品图库"按钮平滑展开，显示产品数量
- **图片按需加载**：产品图片仅在展开图库时才发起网络请求，首屏不加载
- **系列筛选**：全部系列 / 401 / 405 快速切换
- **图片灯箱**：点击产品图片弹出大图，支持左右翻页和键盘操作（Esc关闭、方向键翻页）
- **懒加载背景图**：系列头图通过 IntersectionObserver 接近视口时才加载
- **滚动动画**：元素进入视口时淡入
- **返回顶部按钮**
- **数字计数动画**

## 新增产品系列

后续新增系列只需以下步骤：

1. 在 `assets/images/products/` 下新建文件夹（如 `501/`）
2. 将产品图片复制到该文件夹
3. 在 `index.html` 中参照 401/405 的结构添加新的 `product-series` 区块（包含 series-header、series-toggle、product-grid-wrapper）
4. 在 `main.js` 中添加对应的 `generateFileList501()` 函数并在 `renderProducts()` 中调用
5. 在筛选按钮区域和联系表单的下拉选项中添加新系列

## 图片优化记录

图片使用 macOS `sips` 工具统一转为 JPEG 并压缩：

| 类别 | 原始大小 | 压缩后 | 缩减 |
|------|----------|--------|------|
| 产品图(401) 44张 | 11MB | ~700KB | 94% |
| 产品图(405) 10张 | 2.2MB | ~150KB | 93% |
| 横幅图 | 18MB+ | ~600KB | 97% |
| 工厂图 4张 | 16MB | ~1.4MB | 91% |
| **总计** | **~47MB** | **~3.3MB** | **93%** |

压缩参数：
- 产品图：最长边 600px，JPEG 质量 55%
- 横幅图：最长边 1600-1920px，JPEG 质量 60-70%
- 工厂图：最长边 1200px，JPEG 质量 55%

### 加载优化策略

1. **关键资源预加载**：Hero 背景图和 Logo 通过 `<link rel="preload">` 提前加载
2. **产品图按需加载**：使用 `data-src` 占位，仅在展开图库时才赋值 `src` 发起请求
3. **背景图懒加载**：系列头图通过 `IntersectionObserver` 进入视口 200px 时才加载
4. **原生懒加载**：工厂、关于等区域图片使用 `loading="lazy"`
5. **图片淡入**：加载完成后平滑显示，避免布局抖动
